# 📊 E-Commerce Sales Analytics Dashboard | Power BI Project

## 📌 Project Overview

This project is an interactive E-Commerce Sales Analytics Dashboard developed using Microsoft Power BI. The dashboard provides deep insights into sales performance, customer behavior, delivery analysis, seller performance, and payment trends through visually rich and interactive reports.

The objective of this project is to transform raw e-commerce data into meaningful business insights that help in better decision-making and performance tracking.

---

# 🚀 Features

* 📈 Revenue & Sales Trend Analysis
* 🛒 Orders and Customer Analysis
* 💰 Average Order Value (AOV) Tracking
* 📊 Year-over-Year (YoY) Growth Analysis
* ⭐ Customer Rating & Review Insights
* 🚚 Delivery Performance & Late Order Analysis
* 🌍 State-wise and City-wise Revenue Analysis
* 🏷️ Product Category Performance
* 💳 Payment Method Distribution
* 🧑‍💼 Seller Performance Dashboard
* 🔍 Interactive Filters and Drill-through Reports
